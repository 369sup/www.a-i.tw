# Codex project configuration

This directory contains trusted repository-scoped Codex configuration:

- `config.toml` registers project agents.
- `agents/` contains role briefs.
- `rules/` controls command execution.
- `environments/` describes reproducible project actions.

Personal models, profiles, authentication, notifications, plugin state, lifecycle hooks, MCP launchers, shell paths,
caches, and Desktop settings remain user-managed.
