# Audit and compliance

狀態：Accepted research baseline / no runtime

Audit answers who performed which administrative action on what resource, when, where and with which
result. It is distinct from Activity Feed and Notification.

| Language | Meaning |
| --- | --- |
| Audit event | Immutable administrative observation |
| Audit query | Structured filtering by actor, action, target and time |
| Audit export/stream | Delivery to external assurance systems |
| Compliance report | Evidence package for governance requirements |
| Data residency | Policy determining permitted data region |

Audit events do not replace owner Domain events and cannot become a universal event type.
