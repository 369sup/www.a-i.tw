# Client experience

狀態：Accepted semantic baseline / Web subset current

Web, Mobile, Dashboard, Feed, Inbox, Explore, Command Palette, theme, accessibility, localization,
Markdown authoring and saved replies are experience surfaces or interaction capabilities. They do not
own Account, Repository, Issue, Notification or Search truth.

Presentation adapters map transport and owner-provided results into these surfaces. A view model or
projection remains architecture language and must not become a product Context.
