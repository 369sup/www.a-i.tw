# Language conflicts

狀態：Proposed / empty register

| Term       | Forbidden conflation                                | Canonical owner         | Resolution                                                 |
| ---------- | --------------------------------------------------- | ----------------------- | ---------------------------------------------------------- |
| User       | Principal, Account, credential, owner               | See ubiquitous language | Use the scoped canonical term                              |
| Enterprise | Account, organization, enterprise policy            | Account                 | State whether it is governance scope or resource ownership |
| Role       | organization role, Team maintainer, Repository Role | Account / Repository    | State the scope in every contract                          |
| Access     | authentication success, visibility, grant, decision | Repository              | State relationship, action and resource                    |
| Repository | Git adapter or folder                               | Repository              | Use Repository only for resource container                 |

新增衝突需記錄 source language、target language、owner、translation／ACL 與 removal condition。
