# Naming policy

狀態：Accepted baseline

Use the owner Context's ubiquitous language. Names must state scope: `RepositoryRole` is not `Role`; `AccountId` is not `PrincipalId`; `Credential` is not `Identity`; `Repository Address` is not `RepositoryId`. Do not use `shared`, `common`, `core`, `utils` or `helpers` as an unowned domain dumping ground.
