# 詞彙表

| 詞彙       | 定義                                                |
| ---------- | --------------------------------------------------- |
| Principal  | 可被驗證、可提出操作請求且可供稽核歸因的 actor。    |
| Account    | 擁有資源與命名空間的個人或組織容器。                |
| Namespace  | Account 擁有的人類可讀、可路由名稱。                |
| Repository | 由 Account 擁有的受控協作工作空間與資源政策邊界。   |
| Visibility | 資源 discovery 與基線讀取邊界，不是登入狀態或角色。 |
