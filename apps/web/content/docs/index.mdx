# www.a-i.tw 文件

這是專案的公開產品與工程文件入口。Identity & Access、Account、Repository 已形成
可操作的 in-memory 產品垂直切片；它驗證產品語意與架構邊界，但不代表 production
authentication 或 durable persistence。

從 [開始使用](/docs/getting-started/overview) 了解文件範圍，或直接閱讀
[產品概覽](/docs/product/overview)、[架構模型](/docs/architecture/bounded-contexts) 與
[詞彙表](/docs/reference/glossary)。可在 [產品工作區](/workspace) 操作目前的垂直切片。

內部架構治理、ADR、runbook 與生產證據維護於 repository root 的 `docs/`，不會直接作為公開內容發布。
