---
title: Entitlement
description: Product eligibility versus resource authorization.
---

Generic entitlement is an analytical effective-eligibility concept derived from plan, subscription, license or seat,
membership, enablement, policy, and account state. It is not a universal GitHub top-level object and is distinct from
resource authorization. Support Entitlement is a specifically named enterprise support relationship. Entitlement
runtime is not approved in this repository.
